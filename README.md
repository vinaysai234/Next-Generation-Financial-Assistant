# Next Generation Financial Assistant  
### With Generative AI and Predictive Analytics

The **Next Generation Financial Assistant** is an intelligent web-based personal finance management system designed to help users track income and expenses, analyze spending behavior, predict future expenses, and receive AI-generated financial guidance.

The system combines **Machine Learning**, **Generative AI**, and **Data Analytics** to support smarter financial decision-making through a simple and user-friendly interface.
